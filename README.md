# mppysam
Multiprocessing functionality with pysam

### Install
```
git clone https://github.com/jamesbaye/mppysam.git
cd mppysam
pip install -e .
```

### [dev] Build cython modules
```
python setup.py build_ext --inplace
```

### [dev] Run unittests
In project root folder,
```
python -m unittest
```
